Fair Deal API

A RESTful API, CRUD backend, and demo portfolio app for a fictional mechanic shop built using Node.js, Express.js framework, Sequelize ORM on a MySQL database for persistence. Inspired by minor work I did for my own mechanic.

  • Index page ~ http://localhost:8080/
  • About ~ http://localhost:8080/about
  • Search for Resolved/ Fixed Auto Repair Jobs ~ http://localhost:8080/resolved
